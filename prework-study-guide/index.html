Prework Study Guide

HTML

  • HTML is used to create the structure of a webpage.
  • The head element contains information about the webpage and not visible to the user.
  • The body element represents the visible content shown to the user.

CSS

  • CSS defines the colors, aesthetics, visual layout and organization.
  • There are three ways to style a webpage using CSS: inline CSS, internal CSS style sheet, and external CSS style sheet.
  • Most common way to style a webpage is by using an external CSS style sheet
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Github is used for developing projects and essential for collaboration.
  • git status: checks what branch we are currently on
  • git branch: lists all created branches
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to branch main
  • git pull origin main: adds files from repository to local machine
  • git push origin main: adds updated files from local machine to Github repository
  • git add -A: tracks all unsaved files
  • git commit -m "message": [-m] flag to include a message of the changes implemented
