Git

  • Github is used for developing projects and essential for collaboration.
  • git status: checks what branch we are currently on
  • git branch: lists all created branches
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout main: switches to branch main
  • git pull origin main: adds files from repository to local machine
  • git push origin main: adds updated files from local machine to Github repository
  • git add -A: tracks all unsaved files
  • git commit -m "message": [-m] flag to include a message of the changes implemented
